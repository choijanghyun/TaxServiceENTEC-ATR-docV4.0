### 즉시 조치 (HIGH)

#### 1. 경과규정 추가 (점검 #2)

Section 3.1 또는 3.3에 다음 경과규정을 추가해야 합니다:

```
| 항목 | 내용 |
|------|------|
| 경과규정 | 2015.1.1. 이후 경정청구분부터 5년 적용. 그 이전 청구분은 3년 |
| 근거 | 국세기본법 부칙 (2011.12.31. 법률 제11124호) |
```

현재 시스템이 2026년 기준 최대 5년(2021~2025)을 대상으로 하므로 실무적 영향은 제한적이나, 후발적 경정청구로 2015년 이전 과세연도를 대상으로 할 수 있어 규칙 반영이 필요합니다.


## 구조적 발견사항

### Section 3의 법인세 편향 문제

Section 3 전반이 **법인세 관점**으로 작성되어 있어, 종합소득세 고유의 경정청구 특성이 반영되지 않은 구조적 문제가 있습니다:

| 서브섹션 | 법인세 반영 | 종합소득세 반영 | 비고 |
|---------|:----------:|:-------------:|------|
| 3.1 법적 근거 | O | O (공통) | 처리 관서 기준의 세목별 차이 미확인 |
| 3.2 경정청구 유형 | O | O (공통) | |
| 3.3 법정신고기한 기산일 | O | O | 성실신고확인대상 반영됨 |
| 3.4 가능/불가능 구분 | O (결산조정/신고조정) | **X** (기장 방식별 제약 없음) | 개인사업자 추계 배제 미반영 |
| 3.5 결산확정 원칙 | O ("법인세 전용" 명시) | **X** (대응하는 개인사업자 서브섹션 없음) | |
| 3.6 시스템 구현 요건 | O | ⚠️ (성실신고 판단 REQ 누락) | |

**권고**: Section 3.4와 3.5 사이에 "3.4.1 종합소득세 고유 경정청구 제약 (개인사업자)"를 신설하거나, Section 3.5에 대응하는 "3.5.1 개인사업자의 기장 방식과 경정청구 전략"을 추가할 것을 권장합니다.
