4. **농특세**: 과세/비과세 항목 구분 추가, 근거법령 추가
5. **이월공제**: 이월 기간(10년) 및 이월 가능/불가 구분 추가
6. **상시근로자**: 제외 대상 목록 참조, 법인 합산 산정 방식 언급

### 문서 업데이트 필요

명세서 Section 2 용어 정의를 현재 11개에서 최소 30개(HIGH 19 + 기존 11)로 확대하고, MEDIUM 14개를 추가하면 총 44개로 구성할 것을 권장합니다.
