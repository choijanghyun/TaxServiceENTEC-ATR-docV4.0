## 4. 권장 조치

### 즉시 조치 (HIGH)

1. **과세구조 기본 용어 7개 추가** (종합소득금액, 종합소득 과세표준, 합산과세, 사업소득금액, 비과세소득, 분리과세, 감면 소득 배분 공식)
   - 이 7개는 프롬프트 전체에서 반복 사용되며, 명세서 Section 4(세액 계산 흐름)와 Section 6(항목별 상세)의 모든 공식에 직접 영향

2. **개인사업자 고유 용어 4개 추가** (공동사업자, 손익분배비율, 기장세액공제, 성실신고확인대상)
   - 법인세에는 없는 종합소득세 고유 개념으로, 정의 없이는 Section 6 이해 불가

3. **핵심 감면/공제 명칭 6개 추가** (창업중소기업 세액감면, 중소기업 특별세액감면, 통합투자세액공제, 통합고용세액공제, 연구인력개발비 세액공제, 결손금 소급공제)
   - Section 6~9에서 반복 참조되는 핵심 항목. 약칭/조문번호(제6조, 제7조 등)로도 빈번히 사용되므로 정의에 조문 매핑 필수

4. **세액 계산 프로세스 용어 4개 추가** (환급가산금, 지방소득세, 적용순서, 중소기업)
   - Section 9(환급액 산출 프로세스)의 8단계 계산에 직접 사용

### 문서 업데이트

   - 경정청구: 유형 구분(일반/후발적) 추가

   - 상시근로자: 제외 대상 상세화(대표자 본인 등), 산정 공식 추가
   - 수도권 과밀억제권역: 개인사업자 사업장별 판단 원칙, 하위 구분 명시
