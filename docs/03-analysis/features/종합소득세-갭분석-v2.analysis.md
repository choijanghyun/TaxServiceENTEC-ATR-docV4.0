## 1. 분석 개요

종합소득세 프롬프트 v2.0에 기술된 모든 점검 항목(점검 1 ~ 점검 37)을 요구사항명세서 v2.0과 1:1 대조하여, 각 항목의 반영 수준을 MATCH / PARTIAL / MISSING으로 판정하였다.

### 판정 기준

| 판정 | 의미 |
|------|------|
| **MATCH** | 프롬프트의 핵심 내용이 명세서에 동등 수준으로 반영됨 |
| **PARTIAL** | 일부 반영되었으나 법조항, 계산식, 세부 조건 등이 누락 또는 축약됨 |
| **MISSING** | 명세서에 해당 내용이 전혀 없거나, 확인 불가 수준으로 미흡함 |
