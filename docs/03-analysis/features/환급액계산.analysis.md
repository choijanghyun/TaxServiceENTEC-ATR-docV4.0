## 1. Analysis Overview

### 1.1 Analysis Purpose

법인세 프롬프트(점검 0~32)와 종합소득세 프롬프트(점검 1~37)에 포함된 환급액 계산 관련 모든 내용이 요구사항명세서(v2.0)에 빠짐없이 반영되었는지 체계적으로 검증한다. 분석 관점은 다음 7가지이다:

1. 법인세 점검 항목(0~32) 반영 여부
2. 종합소득세 점검 항목(1~37) 반영 여부
3. 세율/공제율/감면율 수치 정확성
4. 상호배제/최저한세/농특세 처리 로직 완전성
5. 연도별 법령 분기 완전성
6. 법인세/종합소득세 차이 반영
7. 입력 데이터 항목 완전성

### 1.2 Analysis Scope

- **Source (Design)**: 법인세-프롬프트_v2.0.md + 종합소득세-프롬프트_v2.0.md
- **Target (Implementation)**: 환급액계산-요구사항명세서-v2.0.md
- **Analysis Date**: 2026-02-20
