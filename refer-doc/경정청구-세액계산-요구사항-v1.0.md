## 1. 시스템 개요

### 1.1 목적

경정청구 제도를 활용하여 법인사업자(법인세) 및 개인사업자(종합소득세)의 **과다 납부 세액을 환급**받되, AI 기반 점검 로직으로 **환급액을 극대화**하는 시스템

### 1.2 적용 범위

| 구분 | 내용 |
|------|------|
| 세목 | 법인세, 종합소득세 |
| 대상 기간 | 최대 5년 (2026년 기준: 2021~2025 과세연도) |
| 데이터 수집 | 6년치 (경정청구 5년 + 비교용 1년) |
| 대상 납세자 | 중소기업 법인, 개인사업자 (소기업/중기업) |

### 1.3 핵심 가치

- **누락 공제/감면 자동 탐지**: 200여 가지 세액공제/감면 항목 점검
- **최적 조합 산출**: 상호배제 규칙 하 환급액 최대화 조합 자동 선택
- **5년 일괄 분석**: 연도별 법령 차이를 반영한 연속 최적화
- **실질 환급액 기준 판단**: 농특세, 최저한세, 환급가산금 모두 반영

## 3. 세액 계산 로직 전체 구조

### 3.1 법인세 계산 Flow

```
① 각 사업연도 소득 = 익금총액 - 손금총액 (세무조정 반영)
② 과세표준 = 각 사업연도 소득 - 이월결손금 - 비과세소득 - 소득공제
③ 산출세액 = 과세표준 x 법인세율 (4단계 누진세율)
④ (-) 1순위: 세액감면 (조특법 제6조, 제7조)
⑤ (-) 2순위: 이월공제 불가 세액공제
⑥ (-) 3순위: 이월공제 가능 세액공제 (이월분 먼저, 당기분 나중)
     └ 제10조(R&D), 제24조(투자), 제29조의8(고용)
   [최저한세 한도 적용: 과세표준 x 7%/10%/12%/17%]
⑦ (-) 감면분 추가납부세액
⑧ (+) 가산세
⑨ (-) 기납부세액 (중간예납, 원천징수)
= 납부할 세액 (음수이면 환급)
```

**적용순서 근거**: 법인세법 제59조

### 3.2 종합소득세 계산 Flow

```
① 총수입금액 - 필요경비 = 사업소득금액 (사업장별 합산)
② (+) 기타 소득 (근로/이자/배당/연금/기타) = 종합소득금액
③ (-) 소득공제 (인적공제, 연금보험료, 노란우산공제 등) = 과세표준
④ 산출세액 = 과세표준 x 소득세율 (8단계 누진세율)
⑤ (-) 1순위: 세액감면 (제6조 또는 제7조)
⑥ (-) 2순위: 이월공제 불가 세액공제
⑦ (-) 3순위: 이월공제 가능 세액공제 (이월분 먼저, 당기분 나중)
   [최저한세 한도 적용: 산출세액 x 35%/45%]
= 결정세액
⑧ (-) 기납부세액 (중간예납 + 원천징수 + 분납분)
= 차감납부/환급세액
```

**적용순서 근거**: 소득세법 제60조

### 3.3 법인세 vs 종합소득세 핵심 차이

| 구분 | 법인세 | 종합소득세 |
|------|--------|----------|
| 과세단위 | 법인 단위 | 대표자(사업자) 1인 기준 합산과세 |
| 소득공제 | 없음 (이월결손금만) | 인적공제, 연금공제, 노란우산공제 등 |
| 최저한세 기준 | **과세표준** x 7~17% | **산출세액** x 35~45% |
| 세율 | 4단계 (9~24%) | 8단계 (6~45%) |
| 이월결손금 한도 | 중소 100%, 일반 80% | 전액 통산 가능 |
| 감면 소득 배분 | 사업장별 소득비율 | 감면 대상 사업소득 / **종합소득금액** |
| 적용순서 근거 | 법인세법 제59조 | 소득세법 제60조 |
| 소재지 기준 | **본점 간주 규정** (본점이 수도권이면 전 사업장 수도권) | **사업장별** 개별 판단 |
| 개인전용 공제 | 해당없음 | 기장세액공제, 성실사업자 의료비/교육비, 노란우산공제 등 |

### 3.4 4단계 점검 절차 구조

```
[1단계] 환급액 극대화 전략 수립
  └ 그룹 A (감면 중심) vs 그룹 B (공제 중심) 비교 분석

[2단계] 공통 전제 조건 점검
  └ 소재지, 농특세, 최저한세, 중복배제, 중소기업 판단, 적용순서, 세율 등

[3단계] 개별 공제/감면 항목 상세 점검 (32~37개 항목)
  └ 통합투자(제24조) / 통합고용(제29조의8) / 사보(제30조의4)
  └ 창업감면(제6조) / 중소특별감면(제7조) / R&D(제10조)
  └ 법인전용: 외국납부(제57조), 이월결손금 등
  └ 개인전용: 기장세액, 성실사업자 의료비/교육비, 노란우산, 연금저축, 자녀 등

[4단계] 최종 환급액 산출 및 전략 보고
  └ 최적 조합 선택, 환급액 비교표, 필요 증빙, 사후관리 안내
```

## 6. 상호배제 규칙 및 최적 조합 알고리즘

### 6.1 중복적용 매트릭스 (조특법 제127조 제4항)

```
                  제7조   제6조   제24조  제29조의8  제30조의4  제10조
제7조(특별감면)     -      X      X       O        O        O
제6조(창업감면)     X      -      X       ●(주1)   X        O
제24조(투자공제)    X      X      -       O        O        X(주2)
제29조의8(고용)     O      ●      O       -        X(주3)   O
제30조의4(사보)     O      X      O       X(주3)   -        O
제10조(R&D)         O      O      X(주2)  O        O        -

O = 중복 가능, X = 중복 불가
(주1) 2024년까지 중복 가능 / 2025년부터 중복 불가 (택1)
(주2) 동일 자산에 대한 중복 불가
(주3) 2023~24년 선택 적용 (동시 불가)
```

### 6.2 그룹별 전략 비교

| 구분 | 그룹 A (감면 중심) | 그룹 B (공제 중심) |
|------|-------------------|-------------------|
| 핵심 항목 | 제7조(특별감면) 또는 제6조(창업감면) | 제24조(투자) + 제29조의8(고용) |
| 병행 가능 | 제7조 + 제29조의8: 중복 가능 | 투자 + 고용: 중복 가능 |
| 공통 추가 | R&D(제10조)는 어느 그룹과도 병행 | R&D(제10조)는 어느 그룹과도 병행 |
| 장점 | 농특세 면제(실질 100%) | 이월공제 가능(10년) |
| 약점 | 이월공제 불가(감면 소멸) | 농특세 20% 차감(실질 80%) |

### 6.3 최적 조합 알고리즘

```
알고리즘 선택 기준:
- 공제 항목 수 <= 15개: Branch & Bound (최적해 보장)
- 공제 항목 수 > 15개: Greedy (성능 우선)

각 조합에 대해:
  1) 감면/공제 적용
  2) 최저한세 차감
  3) 농특세 차감
  4) netRefund = 실질 환급액 계산
  → 최대 netRefund 조합 선택
```

### 6.4 필수 비교 조합 (4가지)

| 조합 | 산식 | 적합 사례 |
|------|------|----------|
| 1. 제7조+제29조의8+제10조 | 감면세액 + 고용공제x80% + R&D공제 | 고용증가 + R&D + 중소기업 |
| 2. 제6조+제29조의8 (2024이전) | 감면세액 + 고용공제x80% | 창업 5년 이내, 고용 증가 |
| 3. 제24조+제29조의8+제10조 | (투자+고용)x80% + R&D공제 | 투자액 크고 감면 비대상 |
| 4. 제29조의8 단독 | 고용공제x80% | 고용공제만 누락 |

## 8. 환급액 산출 프로세스

### 8.1 8단계 환급금 계산 로직

```
STEP 1: 과세표준 재산정
  [소득세] 총수입금액 - 필요경비 - 소득공제 = 과세표준
  [법인세] 당기순이익 +/- 세무조정 - 이월결손금 = 과세표준

STEP 2: 산출세액 계산
  [소득세] 6~45% 8단계 누진세율
  [법인세] 9~24% 4단계 누진세율

STEP 3: 적용 가능 감면/공제 식별
  업종, 지역, 규모, 기간 등 요건 충족 여부 확인
  적용순서: 감면 -> 이월불가 공제 -> 이월가능 공제

STEP 4: 최적 조합 선택 (상호배제 고려)
  조특법 제127조 제4항 상호배제 규칙 적용
  Branch & Bound / Greedy 알고리즘

STEP 5: 최저한세 적용
  [소득세] 산출세액 x 35%/45%
  [법인세] 과세표준 x 7%/10%/12%/17%

STEP 6: 농어촌특별세 계산
  감면세액 x 20% (과세 항목), 비과세 항목 식별

STEP 7: 환급세액 산출
  환급액 = 기존 결정세액 - 경정 후 결정세액
  ※ 실제 납부세액 초과 불가

STEP 8: 실질환급액 산출
  환급가산금 = 환급세액 x 이자율(~3.1%) x 경과일수/365
  지방소득세 환급 = 본세 환급 x 10%
  실질환급액 = 환급세액 + 환급가산금 + 지방소득세 - 농특세
```

### 8.2 금액 처리 원칙

| 원칙 | 적용 |
|------|------|
| 국세 금액 | 10원 미만 **절사** (반올림 절대 금지, 국세기본법 제86조) |
| 비율/인원수 | 소수점 셋째 자리 **절사**, 둘째 자리까지 표기 |
| 환급가산금 | 1원 미만 절사 |
| 감면 소득 배분 | 분리과세 소득은 분모(종합소득금액)에서 제외 |

### 8.3 환급가산금 계산

```
환급가산금 = 환급세액 x 이자율(연 ~3.1%) x (경과일수 / 365)
- 기산일: 원래 납부일의 다음날
- 종료일: 충당일 또는 지급결정일
- 법인세: 환급가산금은 익금불산입 (법인세법 제18조 제4호)
- 종합소득세: 환급가산금은 비과세 소득
```

### 8.4 최종 보고서 출력 양식

```
[요약표] 적용 가능 공제/감면 항목
  항목명 | 근거법령 | 예상 공제/감면액 | 농특세 | 최저한세 여부 | 실질 환급액

[최적 조합] 1순위/2순위 조합 비교

[환급액 비교표]
  기존 신고 내역 vs 경정 후 내역
  차액(환급액) = 기존 납부세액 - 경정 후 결정세액

[총 수령 예상액]
  법인세/종합소득세 환급액 + 환급가산금 + 지방소득세(10%) - 농특세

[필요 증빙 서류 목록]
[주의사항 및 사후관리]
```

## 부록: 계산 예시

### 법인세 예시

중소기업 법인, 비수도권, 과세표준 5억원, 산출세액 7,500만원, 제7조 감면율 30%, 청년등 2명+일반 1명 증가:

```
① 산출세액: 7,500만원 (= 2억x9% + 3억x19%)
② 감면(제7조): 7,500만원 x 30% = 2,250만원 (농특세 면제)
③ 감면 후 잔여: 5,250만원
④ 최저한세: 5억 x 7% = 3,500만원
   → 감면+공제 합계 한도: 7,500 - 3,500 = 4,000만원
   → 공제 한도: 4,000 - 2,250 = 1,750만원
⑤ 고용공제: (2x1,550) + (1x950) = 4,050만원
   → 최저한세 적용 → 실제 공제 1,750만원 (농특세 350만원)
   → 미공제 2,300만원 이월(10년)
⑥ 실질 환급액: 2,250만원 + (1,750만원 x 80%) = 3,650만원
```

### 종합소득세 예시

중소기업 개인사업자, 비수도권, 산출세액 5,000만원, 제7조 감면율 30%, 소득비율 80%, 청년등 2명+일반 1명 증가:

```
① 산출세액: 5,000만원
② 감면(제7조): 5,000만 x 80% x 30% = 1,200만원 (농특세 면제)
③ 감면 후 잔여: 3,800만원
④ 최저한세:
   - 3천만원분: 3,000만 x 35% = 1,050만원
   - 2천만원분: 2,000만 x 45% = 900만원
   - 합계: 1,950만원
   → 감면+공제 한도: 5,000 - 1,950 = 3,050만원
   → 공제 한도: 3,050 - 1,200 = 1,850만원
⑤ 고용공제: (2x1,550) + (1x950) = 4,050만원
   → 최저한세 적용 → 실제 공제 1,850만원 (농특세 370만원)
   → 미공제 2,200만원 이월(10년)
⑥ 실질 환급액: 1,200만원 + (1,850만원 x 80%) = 2,680만원
```
