## 1. 시스템 개요

### 1.1 목적

경정청구 제도를 활용하여 법인사업자(법인세) 및 개인사업자(종합소득세)의 **과다 납부 세액을 환급**받되, AI 기반 점검 로직으로 **환급액을 극대화**하는 세액 계산 엔진을 구현한다.

### 1.2 적용 범위

| 구분 | 내용 |
|------|------|
| 세목 | 법인세, 종합소득세 |
| 대상 기간 | 최대 5년 (2026년 기준: 2021~2025 과세연도) |
| 데이터 수집 | 6년치 (경정청구 5년 + 비교용 1년) |
| 대상 납세자 | 중소기업 법인, 개인사업자 (소기업/중기업) |
| 경정청구 근거 | 국세기본법 제45조의2 제1항 (법정신고기한 경과 후 5년 이내) |
| 후발적 경정청구 | 판결 확정 등 후발적 사유 발생 시 안 날부터 3개월 이내 (국세기본법 제45조의2 제2항) |
| 2025년 법개정 | 결정·경정 증가세액 경정청구 기한: 90일 → **3개월** 확대 (2025.1.1. 이후 적용, 2024.12.31. 개정) |
| 법령 적용 시점 | 세율·공제율 → 사업연도 **개시일** 기준, 적용기한 → 사업연도 **종료일** 기준, 창업감면율 → **창업 시점** 기준. 세법개정 시행일이 명시된 경우(예: '2025.1.1. 이후 개시 사업연도') **해당 표현 우선**. 모든 항목은 **경정청구 대상 사업연도에 시행 중이던 법령 버전** 기준 판단 (§ [10. 연도별 법령 분기 처리](#10-연도별-법령-분기-처리) 참조) |

### 1.3 핵심 가치

| # | 핵심 가치 | 설명 |
|---|----------|------|
| 1 | 누락 공제/감면 자동 탐지 | 200여 가지 세액공제/감면 항목 점검. 감면신청서 미제출도 경정청구로 적용 가능 (대법원 2003두773, 서면법규과-72) |
| 2 | 최적 조합 산출 | 상호배제 규칙(조특법 제127조 제4항) 하 환급액 최대화 조합 자동 선택 |
| 3 | 5년 일괄 분석 | 연도별 법령 차이를 반영한 연속 최적화 |
| 4 | 실질 환급액 기준 판단 | 농특세, 최저한세, 환급가산금, **지방소득세** 모두 반영 |
| 5 | 법인+개인 통합 | 법인세·종합소득세 양쪽 통합 지원 |

### 1.4 법인세 vs 종합소득세 핵심 차이

| 구분 | 법인세 | 종합소득세 | 근거법령 |
|------|--------|----------|---------|
| 과세단위 | 법인 1개 단위 (사업장 복수여도 법인 합산) | 대표자(사업자) 1인 기준 합산과세 | 법인세법 제2조 / 소득세법 제2조 |
| 소득구조 | 익금−손금 세무조정 (소득공제 없음, 이월결손금만 차감) | 소득합산(사업+근로+이자·배당+연금+기타) 후 소득공제 차감 | 법인세법 제14조 / 소득세법 제14조 |
| 세율 | 4단계 초과누진 9~24% (2023~2025) | 8단계 초과누진 6~45% (과세연도별 구간 상이) | **법인세법 제55조** / **소득세법 제55조** |
| 최저한세 기준 | **과세표준** × 7~17% (중소 7%, 100억↓ 10%, 1,000억↓ 12%, 1,000억↑ 17%) | **산출세액** × 35% (산출세액 3천만원 초과분 45%) | **조특법 제132조** |
| 이월결손금 한도 | 중소 100%, 일반 80% (15년 이월, 2020년 이전 발생분 10년) | 전액 통산 가능 | **법인세법 제13조 제1항** / 소득세법 제45조 |
| 감면 소득 배분 | 사업장별 소득비율로 안분 | 산출세액 × (감면대상 사업소득 ÷ **종합소득금액**) × 감면율 | 조특법 제128조 |
| 적용순서 근거 | 법인세법 제59조 | 소득세법 제60조 | — |
| 소재지 기준 | **본점 간주 규정** (본점이 수도권이면 전 사업장 수도권) | **사업장별** 개별 판단 | 조특법 시행령 별표 |
| 개인전용 공제 | 해당없음 | 기장세액공제, 성실사업자 의료비/교육비, 노란우산공제, 전자신고세액공제, 착한임대인세액공제, 결손금소급공제, 외국납부세액공제 등 | 소득세법 제56조의2, 조특법 제122조의3 등 |
| 법정신고기한 | 사업연도 종료 후 3개월 (12월 결산: 다음 해 3.31) | 다음 해 5월 31일 (성실신고확인대상: 6월 30일) | 법인세법 제60조 / 소득세법 제70조, 제70조의2 |
| 환급가산금 처리 | **익금불산입** | 비과세 소득 | **법인세법 제18조 제4호** |
| 지방소득세 | 국세 경정 확정 후 **별도 경정청구** 필요 | 국세 경정 확정 후 **별도 경정청구** 필요 | **지방세법 제89조** |

#### 1.4.1 종합소득세 고유 특성

법인세에는 없는 종합소득세 고유 과세 구조로, 환급액 계산 시 반드시 반영해야 한다.

| # | 고유 특성 | 상세 내용 | 근거법령 |
|---|----------|----------|---------|
| 1 | **소득합산 과세** | 사업소득 + 근로소득 + 이자·배당소득 + 연금소득 + 기타소득을 합산하여 종합소득금액 산출. 비과세·분리과세 소득(금융소득 2천만원 이하, 분리과세 주택임대소득 등)은 제외하되, 금융소득 2천만원 초과 시 전액 종합과세 포함 | 소득세법 제14조, 제16조 |
| 2 | **공동사업자 소득배분** | 공동사업장의 소득은 손익분배비율에 따라 각 공동사업자에게 배분 후 각자의 종합소득에 합산. 감면 대상 소득도 본인 지분(손익분배비율)에 해당하는 소득에 대해서만 적용 | **소득세법 제43조** |
| 3 | **감면 소득 배분 공식** | 감면세액 = 산출세액 × (감면 대상 사업소득 ÷ 종합소득금액) × 감면율. 분모인 '종합소득금액'에서 비과세·분리과세 소득은 제외 | 조특법 제128조 |
| 4 | **기장세액공제 특례** | 소득세법 제56조의2에 따른 기장세액공제는 조특법 제127조 중복배제 대상이 **아님** → 모든 조특법상 감면·공제와 병행 가능. 간편장부대상자가 복식부기로 기장한 경우에도 적용 | **소득세법 제56조의2** |
| 5 | **성실신고확인대상** | 업종별 수입금액 기준 초과 시 성실신고확인대상에 해당하며 법정신고기한이 6월 30일로 연장. 성실신고확인비용 세액공제(60%, 연 120만원 한도) 적용 가능 | 소득세법 제70조의2, 조특법 제126조의6 |
| 6 | **결손금 소급공제** | 중소기업인 개인사업자가 해당 과세연도에 결손금 발생 시, 직전 과세연도 납부세액을 한도로 소급공제 환급 신청 가능 | **소득세법 제85조의2** |

## 2. 용어 정의

### 2.1 기본 과세체계 용어

| 용어 | 정의 |
|------|------|
| 경정청구 | 세금을 과다 납부한 경우 관할 세무서장에게 환급을 요청하는 제도. ①일반 경정청구: 법정신고기한 경과 후 5년 이내 (국세기본법 제45조의2 제1항), ②후발적 경정청구: 판결 확정 등 후발적 사유 발생 시 안 날부터 3개월 이내 (동조 제2항) |
| 과세표준 | 세율을 적용하여 세액을 산출하는 기준 금액. 법인: 각 사업연도 소득 − 이월결손금 − 비과세소득 − 소득공제 (법인세법 제13조). 개인: 종합소득금액 − 소득공제 (소득세법 제14조) |
| 각 사업연도 소득 | 법인의 해당 사업연도에 속하는 익금총액에서 손금총액을 차감한 금액 (법인세법 제14조). 세무조정 후 확정 |
| 종합소득금액 | 이자·배당·사업·근로·연금·기타소득을 합산한 금액. 비과세·분리과세 소득 제외, 금융소득 2천만원 초과 시 전액 포함 (소득세법 제14조, 제16조) |
| 산출세액 | 과세표준에 세율(초과누진세율)을 적용하여 산출한 세액. 법인: 법인세법 제55조, 개인: 소득세법 제55조 |
| 결정세액 | 산출세액에서 세액공제·세액감면을 차감한 최종 확정 세액 |
| 납부세액 | 결정세액에서 기납부세액(중간예납·원천징수·수시부과)을 차감한 실제 납부(또는 환급) 금액 |
| 이월결손금 | 과거 사업연도에 발생한 결손금으로 이후 과세연도에 이월하여 소득에서 공제하는 금액. 법인: 15년 이월, 중소 100%·일반 80% 한도 (법인세법 제13조 제1항, 2020년 이전 발생분 10년) |
| 익금 | 법인의 순자산을 증가시키는 거래로 인한 수익 (법인세법 제15조). 자본거래 제외 |
| 손금 | 법인의 순자산을 감소시키는 거래로 인한 비용·손실 (법인세법 제19조). 자본거래 제외 |
| 세무조정 | 기업회계상 당기순이익을 세법 기준으로 조정하여 각 사업연도 소득을 산출하는 과정 (손금불산입·익금산입·손금산입·익금불산입) |
| 초과누진세율 | 과세표준을 구간별로 나누어 각 구간에 해당하는 세율을 적용하는 방식. 법인 4단계 (9~24%), 개인 8단계 (6~45%) |

### 2.2 경정청구·신고 관련 용어

| 용어 | 정의 |
|------|------|
| 결산조정 | 결산서(재무제표)에 비용으로 계상해야만 세무상 손금으로 인정되는 항목 (감가상각비, 퇴직급여충당금 등). 결산서 미계상 시 경정청구로 소급 적용 **불가** |
| 신고조정 | 결산서 변경 없이 세무조정계산서 또는 신고서에서만 조정하여 적용하는 항목 (세액감면·세액공제 신규 적용, 익금불산입 누락 수정 등). 경정청구로 소급 적용 **가능** |
| 결산확정의 원칙 | 법인세 과세표준은 결산서에 의하여 확정된 것을 기초로 하는 원칙 (법인세법 제40조). 결산조정 항목은 결산서를 변경하지 않는 한 경정청구 불가 |
| 후발적 경정청구 | 판결 확정, 경정결정 등 후발적 사유 발생 시 그 사유를 안 날부터 3개월 이내 청구 (국세기본법 제45조의2 제2항) |
| 법정신고기한 | 법인세: 사업연도 종료 후 3개월 (법인세법 제60조). 종합소득세: 다음 해 5.31 (소득세법 제70조), 성실신고확인대상: 6.30 (소득세법 제70조의2) |
| 적용순서 | 세액감면·세액공제를 산출세액에서 차감하는 법정 순서. 법인: 법인세법 제59조, 개인: 소득세법 제60조 |

### 2.3 공제·감면 관련 용어

| 용어 | 정의 |
|------|------|
| 세액감면 | 산출세액에서 일정 비율을 면제하는 것 (예: 창업중소기업 감면 50~100%). 감면율 × 감면 대상 소득 비율 적용 |

### 3.1 법적 근거

| 항목 | 내용 |

|---|----------|------|
| 1 | 감가상각비 추가 계상 | 결산서에 비용 미계상 시 소급 불가 |
| 2 | 퇴직급여충당금 추가 설정 | 결산조정 필수 |
| 3 | 대손충당금 추가 설정 | 결산조정 필수 |
| 4 | 감가상각 방법 변경 | 별도 신고 절차 필요 |
| 5 | 결산서(재무제표)를 변경해야 하는 모든 항목 | 결산확정의 원칙 (법인세법 제40조) |

#### 3.4.3 경정청구 자체가 불가능한 경우 ★★★

| # | 불가능 사유 | 근거법령 | 시스템 구현 |
|---|----------|---------|-----------|
| 1 | 법정신고기한 후 **5년 경과** | 국세기본법 제45조의2 제1항 | 기한 자동 계산 → 접수 차단 |
| 2 | **결산확정의 원칙**에 반하는 결산 변경 | 법인세법 제40조 | 결산조정 항목 포함 시 경고 |
| 3 | 과세관청 **직권경정** 시 과소신고분에 대한 감면 적용 | **조특법 제128조 제3항** | 직권경정 이력 확인 → 해당 연도 감면 적용 차단 |

> **REQ-3.4.1**: 시스템은 경정청구 접수 전 위 3가지 "자체 불가" 조건을 자동 판단하여 사전 차단해야 한다.

### 3.5 결산확정의 원칙과 경정청구 전략 (법인세 전용, REQ-3.5)

> **근거법령**: 법인세법 제40조 (결산확정 기준)

법인세법상 '결산조정' 항목과 '신고조정' 항목의 구분이 경정청구 전략의 핵심이다.

|------|------|:----------------:|------|

| **신고조정 항목** | 결산서에 미반영해도 세무조정으로 손금 인정 가능 | **가능** | 일시상각충당금, 압축기장충당금, 국고보조금 익금불산입 |

### 3.5.1 개인사업자의 기장 방식과 경정청구 전략 (종합소득세 전용, REQ-3.5.1)

개인사업자는 기장 방식(복식부기/간편장부/추계)에 따라 경정청구 적용 가능 범위가 달라진다.

| 기장 방식 | 감면·공제 적용 | 경정청구 가능 범위 | 비고 |

| **간편장부** | 대부분 적용 가능 | 신고조정 항목 + 기장세액공제(복식부기 전환 시) | 복식부기 전환 유도 검토 |
| **추계신고** | **대부분 적용 불가** | 세액감면·세액공제 원칙적 배제 (조특법 제128조) | ★ 추계→기장 전환 후 수정신고 검토 |

**시스템 구현 요건:**

| REQ-ID | 요건 | 우선순위 |
|--------|------|:-------:|
| REQ-3.5.1.1 | 입력 시 기장 방식(복식부기/간편장부/추계) 확인 → 추계신고 시 감면·공제 적용 자동 차단 및 경고 | 필수 |
| REQ-3.5.1.2 | 추계신고 사업자의 경우 기장 전환 후 수정신고 → 경정청구 전략을 대안으로 안내 | 권장 |
| REQ-3.5.1.3 | 간편장부대상자가 복식부기로 기장한 경우 기장세액공제 자동 적용 여부 점검 | 필수 |

### 4.1 법인세 계산 Flow

```
REQ-4.1: 법인세 산출세액 계산 파이프라인

① 각 사업연도 소득 = 익금총액 − 손금총액 (세무조정 반영)
   ├ 세무조정 재검토: 접대비(제25조)/업무무관자산 차입금이자(제28조)/
   │  가지급금 인정이자(시행령 §89)/수입배당금 익금불산입(§18의2)/
   │  업무용 승용차(§27의2, 시행령 §50의2) 오류 점검
   └ ★ 세무조정 오류 수정 → 과세표준 변동 → 산출세액 변동 → 환급 가능

② 이월결손금 공제 (법인세법 제13조 제1항)
   ├ 중소기업: 각 사업연도 소득의 100% 한도
   ├ 중소기업 외: 각 사업연도 소득의 80% 한도
   ├ 이월 기간: 15년 (2020년 이전 발생분 10년)
   └ ★ [연쇄 변동] 경정청구로 소득 변동 시:
      과세표준 변동 → 산출세액 변동 → 공제·감면 효과 변동 → 최저한세 재산정

② -1 결손금 소급공제 분기 (법인세법 제72조, 중소기업 한정)
   ├ 해당 사업연도 결손금 발생 시 직전 사업연도 납부세액 한도 소급환급
   ├ 이월공제 vs 소급공제 실익 비교 (현재가치 기준)
   └ 판단 기준: 향후 과세소득 전망, 현금 흐름 필요성, 이월결손금 소멸 위험

③ 과세표준 = 각 사업연도 소득 − 이월결손금 − 비과세소득 − 소득공제

④ 산출세액 = 과세표준 × 법인세율 (법인세법 제55조, 4단계 초과누진)
   ├ ~2022년: 2억↓ 10%, 2~200억 20%, 200~3,000억 22%, 3,000억↑ 25%
   ├ 2023~2025년: 2억↓ 9%, 2~200억 19%, 200~3,000억 21%, 3,000억↑ 24%
   └ 2026년~: 해당 사업연도 시행 법령 기준 확인 필수

⑤ (+) 토지 등 양도세 추가과세 (법인세법 제55조의2)
   └ 적용 오류 시 경정청구 대상

⑥ (-) 1순위: 세액감면 (조특법 제6조, 제7조 등)
⑦ (-) 2순위: 이월공제 불가 세액공제
⑧ (-) 3순위: 이월공제 가능 세액공제 (이월분 먼저, 당기분 나중)
     └ 제10조(R&D), 제24조(투자), 제29조의8(고용) 등
   ┌─────────────────────────────────────────────┐
   │ [최저한세 한도] 조특법 제132조                      │
   │  과세표준 × 7%(중소) / 10%(100억↓) / 12%(1,000억↓) / 17%(1,000억↑)  │
   │                                                   │
   │ [적용순서 규칙] 조특법 제132조 제3항, 시행령 제126조 제5항  │
   │  · 최저한세 배제 항목 먼저 공제 → 대상 항목 나중 공제     │
   │  · 납세자가 배제 항목을 선택할 권리 보유               │
   │  · 과세관청 경정 시: 간접감면 우선 배제                │
   └─────────────────────────────────────────────┘

⑨ (-) 감면분 추가납부세액 (감면취소 시)
⑩ (+) 가산세
⑪ (-) 기납부세액 (중간예납 + 원천징수 + 수시부과)
= 결정세액 (차감납부 또는 환급)

⑫ (-) 농어촌특별세 (감면세액의 20%, 비과세 항목 제외)
     └ 비과세: 제6조·제7조 감면, 제10조 R&D 공제 등 (농특세법 제4조)
⑬ (-) 지방소득세 (법인세의 10%, 별도 경정청구 필요, 지방세법 제89조)
= 실질 환급액
```

```

   └ 금융소득 2천만원 초과 시: 전액 종합과세 포함

   └ ★ 과세연도별 세율 구간이 다르므로 경정청구 대상 연도에 맞는 세율 적용 필수

⑤ (-) 1순위: 세액감면 (조특법 제6조 또는 제7조 등)
   ├ [감면 소득 배분 공식] (소득세법 시행령 제117조, 조특법 제128조)
   │  감면세액 = 산출세액 × (감면 대상 사업소득 ÷ 종합소득금액) × 감면율
```

**상시근로자 증가 인원 산정 로직 (REQ-6.1.4.A):**

```
STEP 1: 전체 증가분 = 해당연도 전체 - 직전연도 전체
        → 증가분 <= 0이면 공제 불가 (종료)
STEP 2: 청년등 증가분 = 해당연도 청년등 - 직전연도 청년등
        → <0이면 0, >전체 증가분이면 전체로 한정
STEP 3: 일반 증가분 = 전체 증가분 - 청년등 증가분
        → <0이면 0
STEP 4: 소수점 셋째 자리 절사, 둘째 자리까지 표기
```

| 지분율 | 익금불산입률 |
|--------|:----------:|
| 50% 이상 | **100%** |
| 30% 이상 ~ 50% 미만 | **80%** |
| 30% 미만 | **30%** |

> **경과규정**: 2023~2024년 배당분은 종전 규정 적용도 허용 → 유리한 쪽 선택

**2022.12.31. 이전 수취 배당분** (구법 기준):
→ 상장·비상장·지주회사 여부에 따라 4~5단계 구분

| 지분율 | 익금불산입률 |
|--------|:----------:|
| 100% (완전 자회사) | **전액** |
| 80% 이상 | **80%** |
| 40% 이상 | **60%** |
| 20% 이상 | **30%** |
| 그 외 | **0%** (전액 익금산입) |

> **주의**: 구법은 상장법인·비상장법인·지주회사별 세부 기준이 상이. 경정청구 대상 사업연도에 맞는 법령 버전 확인 필수.
> 지분 변동이 반영되지 않은 경우 → 과세표준 변동 → 환급 가능.

### 7.3 최적 조합 알고리즘 (REQ-7.3)

```
REQ-7.3: 최적 조합 탐색 알고리즘

알고리즘 선택 기준:
- 공제 항목 수 <= 15개: Branch & Bound (최적해 보장)
- 공제 항목 수 > 15개: Greedy (성능 우선)

function findOptimalCombination(credits, exclusionRules):
    validCombinations = generateValidCombos(credits, exclusionRules)
    bestRefund = 0
    bestCombo = null

    for each combo in validCombinations:
        1) 감면/공제 적용 (적용순서 준수)
        2) 최저한세 적용 → 초과분은 이월공제 대상 확인
        3) 농특세 차감 (항목별 과세/비과세 구분)
        4) netRefund = 실질 환급액 계산

        if netRefund > bestRefund:
            bestRefund = netRefund
            bestCombo = combo

    return bestCombo
```

## 9. 환급액 산출 프로세스 (8단계)

### 9.1 8단계 환급금 계산 로직 (REQ-9.1)

```
STEP 1: 과세표준 재산정
  [소득세] 총수입금액 - 필요경비 - 소득공제 = 과세표준
  [법인세] 당기순이익 +/- 세무조정 - 이월결손금 = 과세표준

  ★ [소득세 전용] 소득공제 극대화 전략 (세율 구간 최적화):
    개인사업자는 소득공제 항목이 다양하므로(인적공제, 연금보험료,
    노란우산공제 등) 소득공제를 먼저 최대한 반영하여 '과세표준'
    자체를 낮추는 것이 핵심 전략.
    → 과세표준이 낮아지면 적용 세율 구간이 변동될 수 있으며,
      산출세액이 감소하여 세액감면·세액공제 효과도 달라짐.
    → 필수 순서:
      ① 소득공제 항목을 모두 반영하여 과세표준 확정
      ② 변동된 과세표준에 해당하는 세율 구간 확인
      ③ 산출세액 재산정 (STEP 2)
      ④ 재산정된 산출세액 기준으로 세액감면·세액공제 적용 (STEP 3~4)
    → 예시: 노란우산공제 500만원 추가 → 과세표준 8,800만→8,300만
      → 세율 35%→24% 구간 이동 → 산출세액 대폭 감소
    → ※ 소득공제 반영 전후의 산출세액 차이를 환급액 비교표에 반드시 포함

STEP 2: 산출세액 계산
  [소득세] 6~45% 8단계 누진세율 (과세연도별 세율 적용)
  [법인세] 9~24% 4단계 누진세율 (과세연도별 세율 적용)

STEP 3: 적용 가능 감면/공제 식별
  업종, 지역, 규모, 기간 등 요건 충족 여부 확인
  적용순서: 감면 → 이월불가 공제 → 이월가능 공제

STEP 4: 최적 조합 선택 (상호배제 고려)
  조특법 제127조 제4항 상호배제 규칙 적용
  Branch & Bound / Greedy 알고리즘

STEP 5: 최저한세 적용
  [소득세] 산출세액 x 35%/45%
  [법인세] 과세표준 x 7%/10%/12%/17%

STEP 6: 농어촌특별세 계산
  감면세액 x 20% (과세 항목), 비과세 항목 식별

STEP 7: 환급세액 산출
  환급액 = 기존 결정세액 - 경정 후 결정세액
  ※ 환급액은 실제 납부세액(본세+중간예납+원천징수 등)을 초과할 수 없음
  ※ [법인] 분납 이력이 있는 경우 분납 납부일별 환급가산금 기산점이 다름
  ※ [법인] 중간예납 과납분이 확정신고 시 미환급된 경우 함께 정리
  ※ 시스템은 기존 신고 내역의 실납부세액 총액을 확인하여 환급한도로 설정

STEP 8: 실질환급액 산출
  환급가산금 = 환급세액 x 이자율(~3.1%) x 경과일수/365
  지방소득세 환급 = 본세 환급 x 10%
  실질환급액 = 환급세액 + 환급가산금 + 지방소득세 - 농특세
```

## 16. 계산 예시

### 16.1 법인세 계산 예시

> 중소기업 법인, 비수도권, 과세표준 5억원, 산출세액 7,500만원
> 제7조 감면율 30%, 청년등 2명+일반 1명 증가

```
① 산출세액: 7,500만원 (= 2억x9% + 3억x19%)
② 감면(제7조): 7,500만원 x 30% = 2,250만원 (농특세 면제)
③ 감면 후 잔여: 5,250만원
④ 최저한세: 5억 x 7% = 3,500만원
   → 감면+공제 합계 한도: 7,500 - 3,500 = 4,000만원
   → 공제 한도: 4,000 - 2,250 = 1,750만원
⑤ 고용공제: (2x1,550) + (1x950) = 4,050만원
   → 최저한세 적용 → 실제 공제 1,750만원 (농특세 350만원)
   → 미공제 2,300만원 이월(10년)
⑥ 실질 환급액: 2,250만원 + (1,750만원 x 80%) = 3,650만원
```

### 16.2 종합소득세 계산 예시

> 중소기업 개인사업자, 비수도권, 산출세액 5,000만원
> 제7조 감면율 30%, 소득비율 80%, 청년등 2명+일반 1명 증가

```
① 산출세액: 5,000만원
② 감면(제7조): 5,000만 x 80% x 30% = 1,200만원 (농특세 면제)
③ 감면 후 잔여: 3,800만원
④ 최저한세:
   - 3천만원분: 3,000만 x 35% = 1,050만원
   - 2천만원분: 2,000만 x 45% = 900만원
   - 합계: 1,950만원
   → 감면+공제 한도: 5,000 - 1,950 = 3,050만원
   → 공제 한도: 3,050 - 1,200 = 1,850만원
⑤ 고용공제: (2x1,550) + (1x950) = 4,050만원
   → 최저한세 적용 → 실제 공제 1,850만원 (농특세 370만원)
   → 미공제 2,200만원 이월(10년)
⑥ 실질 환급액: 1,200만원 + (1,850만원 x 80%) = 2,680만원
```
